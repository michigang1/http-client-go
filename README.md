# Git http server

This is lab 1 for software architecture subject.

## Installation

### 1. Clone repository

```bash
git clone https://github.com/MrPaschenko/http-client-go.git
```

### 2.  Run

```bash
go run *.go
```

## Using

Make get request to `localhost:8795/time` to get current time.
